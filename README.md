# vue-chat

> 一个聊天界面，基于[vuejs2.0](http://vuejs.org/)、[mint-ui](http://mint-ui.github.io/).

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
